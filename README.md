## Glossário

- True: истинный
- False: ложь
- Integer: целое число
- String: строка
- Decimal: десятичная дробь
- Boolean: логический
- Do: делать
- While: пока
- For: для
- Function: функция
- Return: вернуть
- If: если
- Else: иначе

## Expressões Regulares

- INT -> ```[0-9]+```
- DEC -> ```[0-9]+.[0-9]+```
- ID  -> ```[a-zA-Z\u0400-\u04FF]+```
- STRING -> ``` '[a-zA-Z\u0400-\u04FF_*/@=^<>!...]+' ```
- BOOLEAN -> ``` 'истинный | ложь' ```
- TYPE -> ``` 'целое число' | 'десятичная дробь' | 'строка' | 'логический' ```
- PLUS -> ``` '+' ```
- MINUS -> ``` '-' ```
- TIMES -> ``` '*' ```
- DIV -> ``` '/' ```
- POW -> ``` '^' ```
- MOD -> ``` '%' ```
- ASSIGN -> ``` := ```
- PLUS_ASSIGN -> ``` '+:=' ```
- MINUS_ASSIGN -> ``` '-:=' ```
- TIMES_ASSIGN -> ``` '*:=' ```
- POW_ASSIGN -> ``` '^:=' ```
- GREATER -> ``` '>' ```
- LESS -> ``` '<' ```
- GREATER_EQUAL -> ``` '>=' ```
- LESS_EQUAL -> ``` '<=' ```
- EQUAL -> ``` '==' ```
- DIFFERENT -> ``` '!=' ``` 
- AND -> ``` '&' ```
- OR -> ``` '||' ```
- OPEN_BRACKET -> ``` '(' ```
- CLOSE_BRACKET -> ``` ')' ```
- OPEN_PARENTHESIS -> ``` '{' ```
- CLOSE_PARENTHESIS -> ``` '}' ```
- OP_IF -> ``` 'если' ```
- OP_ELSE -> ``` 'иначе' ```
- OP_DO -> ``` 'делать' ```
- OP_WHILE -> ``` 'пока' ```
- OP_FOR -> ``` 'для' ```
- OP_FUNCTION -> ``` 'функция' ``` 
- OP_RETURN -> ``` 'вернуть' ```
- EOF -> ``` '$' ```
