## Tokens

- INT -> ```[0-9]+```
- DEC -> ```[0-9]+.[0-9]+```
- ID  -> ```[Aa-Zz]+```
- TYPE -> ``` 'int' | 'decimal' | 'string' | 'boolean' ```
- PLUS -> ``` '+' ```
- MINUS -> ``` '-' ```
- TIMES -> ``` '*' ```
- DIV -> ``` '/' ```
- POW -> ``` '^' ```
- MOD -> ``` '%' ```
- ASSIGN -> ``` := ```
- PLUS_ASSIGN -> ``` '+:=' ```
- MINUS_ASSIGN -> ``` '-:=' ```
- TIMES_ASSIGN -> ``` '*:=' ```
- POW_ASSIGN -> ``` '^:=' ```
- GREATER -> ``` '>' ```
- LESS -> ``` '<' ```
- GREATER_EQUAL -> ``` '>=' ```
- LESS_EQUAL -> ``` '<=' ```
- EQUAL -> ``` '==' ```
- DIFFERENT -> ``` '!=' ``` 
- AND -> ``` '&' ```
- OR -> ``` '||' ```
- OPEN_BRACKET -> ``` '(' ```
- CLOSE_BRACKET -> ``` ')' ```
- OPEN_PARENTHESIS -> ``` '{' ```
- CLOSE_PARENTHESIS -> ``` '}' ```
- OP_IF -> ``` 'IF' ```
- OP_ELSE -> ``` 'ELSE' ```
- OP_DO -> ``` 'DO' ```
- OP_WHILE -> ``` 'WHILE' ```
- OP_FOR -> ``` 'FOR' ```
- OP_FUNCTION -> ``` 'FUNC' ``` 
- OP_RETURN -> ``` 'RETURN' ```
- EOF -> ``` '$' ```
